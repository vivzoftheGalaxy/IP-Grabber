Documentation for the Python Script

Overview

This Python script performs several tasks, including:

  • Installing necessary libraries (requests and keyboard) if they are not already installed.
  • Retrieving and displaying the public and private IP addresses of the machine.
  • Providing a user interface to toggle the display of IP addresses using keyboard inputs.

The script is designed to be executed from the command line or a Python environment that supports keyboard input.

Functions

install_library(library_name)

Purpose: Installs the specified Python library using pip.

Parameters:

  • library_name (str): The name of the library to be installed.

Usage:

install_library('requests')

get_public_ip()

Purpose: Fetches the public IP address of the machine using the ipify API.

Returns:

  • str: The public IP address or an error message if the request fails.

Usage:

public_ip = get_public_ip()

get_private_ip()

Purpose: Retrieves the private IP address of the machine.

Returns:

  • str: The private IP address or an error message if the retrieval fails.

Usage:

private_ip = get_private_ip()

toggle_prompt()

Purpose: Provides a simple user interface to toggle between options using arrow keys and select using the Enter key.

Returns:

  • bool: True if the user selects 'y' (Yes), False if the user selects 'n' (No).

Usage:

show_ip = toggle_prompt()

  1. Print the Script Header: Displays a header line for visual separation.

  2. Library Imports and Installation: Attempts to import requests and keyboard. If the import fails, it installs the library using pip and then re-imports it.

    import requests
except ImportError:
    install_library('requests')
    import requests

try:

    install_library('keyboard')

  5. Display IP Addresses: If the user chooses to display IP addresses, the script fetches the public and private IP addresses and prints them.

Example Usage

To run this script, save it as a .py file and execute it in a terminal or command prompt:

python script_name.py

Follow the on-screen instructions to toggle and display your IP addresses.

Notes

  • Ensure you have internet access to retrieve the public IP address using the ipify API.
  • The script uses ANSI escape codes for coloring text in the terminal, which might not be supported in all environments.
